# VBA_Stock_Data

## Ojective

Use VBA to analysis stock data information.

## Purpose

To analysis stock data using VBA to calculate for all stocks
- yearly change
- % change
- total volume
- Apply conditional formatting for positive and negative changes

## Technologies Used

- VBA
